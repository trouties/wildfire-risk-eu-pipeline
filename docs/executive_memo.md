## Methodology Summary

Buildings are scored using a weighted composite index across five risk groups:

| Risk Group | Weight | Key Drivers |
|------------|--------|-------------|
| Vegetation | 30% | NDVI, forest proximity, WUI class |
| Terrain | 20% | Slope, south-facing aspect |
| Fire Weather | 20% | Fire Weather Index (FWI) statistics 2015–2024 |
| Fire History | 20% | Proximity to historical fire perimeters, VIIRS hotspot density |
| Proximity | 10% | Distance to fire station, road access |

Buildings are scored and classified into quintiles (Very Low to Very High), each containing approximately 20% of the assessed portfolio.

## Key Limitations

1. **Suburban encroachment blind spot**: Fires spreading from wildland into low-vegetation suburban areas (Acharnes archetype) are outside the model's design envelope. v2 dynamic features do not address this fire type.

2. **4-event LOEO**: Minimally viable for cross-validation. Expanding to 6+ events requires AOI extension beyond Attica.

3. **FWI resolution**: Fire Weather Index data is available at ~9 km grid resolution. Buildings within the same grid cell receive identical weather scores regardless of micro-topographic differences.

4. **Data vintage**: CORINE Land Cover 2018 is used for vegetation classification. Recent land-use changes (e.g., post-fire regeneration, new construction) are not reflected.
