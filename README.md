# WildfireRisk-EU

Asset-level wildfire risk scoring for Mediterranean wildland-urban interface (WUI) buildings.

A reproducible geospatial data pipeline that scores buildings in the Athens WUI for wildfire exposure using open data sources (OSM, Copernicus, ERA5, EFFIS, FIRMS).

## Quick Start

```bash
conda env create -f environment.yml
conda activate geo311
pip install -e ".[dev]"
```
